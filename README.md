# Dubai to Durban Perfume Store

A modern ecommerce frontend built with Next.js and Supabase.

## Features

- Product listing from Supabase database
- Add to cart functionality
- Remove from cart
- Dynamic total calculation
- Dark / Light theme toggle
- Responsive UI design

## Tech Stack

- Next.js (React)
- Tailwind CSS
- Supabase (Database)
- Vercel (Deployment)

## Setup

1. Clone repository
2. Install dependencies

npm install

3. Run development server

npm run dev

## Database

Products are stored in Supabase.

Fields:

- id
- name
- price
- image

## Deployment

Deployed via Vercel

## Notes

- Product images are stored in `/public/products`
- Prices can be updated directly in Supabase
